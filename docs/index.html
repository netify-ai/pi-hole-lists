Application Lists for Pi-Hole

Netify Applications Lists for Pi-hole

Are you a Pi-hole user looking to whitelist or blacklist a particular application like TikTok, SnapChat, etc? If so, read on.

What is an Application?

Basically, an application is a defined grouping of base domains. To give you an example, let's start with a small sample of hostnames used by TikTok:

  • pull-rtmp-l1-va01.tiktokcdn.com
  • hotapi-va.tiktokv.com
  • tiktok.bytedance.map.fastly.net

The bolded part of the hostname is part of the TikTok application definition. So any hostname matching ANYTHING.tiktokcdn.com is consider part of TikTok.

Application Lists

You can find the application lists organized by category in the adjacenct menu. You can also get the whole list by clicking on the Everything link.

Getting Started

A Pi-hole application list uses the Regex feature. At this time, these definitions cannot be loaded via the Adlists feature. To get started, take a look at the application for Twitter below:

Twitter

Twitter is an online social networking service that allows users to post 140-character messages to other users and general public. 

(\.|^)ads-twitter\.com$
(\.|^)t\.co$
(\.|^)twimg\.com$
(\.|^)twitter\.com$
(\.|^)twitter\.map\.fastly\.net$
(\.|^)twttr\.com$

If you wanted to blacklist or whitelist Twitter, you can add the regex to Pi-hole by doing the following:

  • Log into the Pi-hole admin panel
  • Click on Group Management > Domains in the menu
  • Click on the RegEx filter tab
  • Cut and paste the data from this site into the Regular Expression input box
  • Add Twitter to the comment to keep track
  • Click on the Add button for either blacklist or whitelist
